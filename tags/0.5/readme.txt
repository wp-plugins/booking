Booking calendar for making Reservation at your site and blogs. 
== Description ==

Using this plugin, you can add booking service to your site. Your site visitors can make booking of one or several days. Its can be interesting for hotel reservation service, rental service or any other service, where is needed making reservation at specific dates.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload entire `wpdev-booking` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The Plugin menu "Booking" will appear atadmin menu
4. Add bookmark to  your post in format [booking]  (You can use parameters numMonths='1' for changing numbers of month at the page.)

== Frequently Asked Questions ==

= How to add Booking form to my post? =

Just insert this bookmark [booking numMonths='1'] to thepost.

= How to change numbers of months at my post? =

At bookmark [booking numMonths='1'] of your post change parameter 'numMonths'
